# Product Analytics Dashboard

This project analyzes user events and product engagement metrics.

## Business Questions
- Where do users drop in the funnel?
- What features drive repeated usage?
- How does retention vary by user segment?

## SQL Scripts
- `create_tables.sql` → Create table and load CSV
- `funnel_analysis.sql` → Track users through funnel stages
- `retention_analysis.sql` → Retention over time
- `feature_adoption.sql` → Feature usage analysis

## Data
- `product_events.csv` → Event-level data including date, device type, marketing channel, page views, add-to-cart, purchases
